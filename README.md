## TYLES AUSTRIA BROWSER PLUGIN

Hallo lieber Pixler für das folgende script benötigst du das **browser-plugin Tampermoneky**.

In deinen browser einstellungen unter "plugins" suche im store nach **Tampermonkey**, und **installiere** das **plugin**!

Unter Chrome sieht das so aus:

Gehe auf ... -> Extensions -> Visit Chrome Web Store :

<img width="1616" height="942" alt="image" src="https://github.com/user-attachments/assets/57aeb005-bb7c-4e97-95fa-797a0652f392" />

Jetzt suche in der suchleiste Tampermonkey:

<img width="2552" height="1482" alt="image" src="https://github.com/user-attachments/assets/2f30416b-1731-4244-9406-109213cadc21" />

Klicke auf Tampermonkey dann öffnet sich ein neues Fenster:

<img width="2556" height="782" alt="image" src="https://github.com/user-attachments/assets/ec6c67b4-3a79-4237-82d9-9549e552fdd9" />

Klicke hier auf Add to Chrome.

Es öffnet sich ein weiteres fenster was nach berechtigungen fragt akzeptiere diese:

<img width="980" height="568" alt="image" src="https://github.com/user-attachments/assets/9ef88d75-346e-4392-970e-fa54a7e692fe" />
